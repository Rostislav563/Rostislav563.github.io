# 😊 Emotion Detector

Веб-приложение для распознавания эмоций на лицах в реальном времени с использованием машинного обучения.

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript)
![Vite](https://img.shields.io/badge/Vite-8-646CFF?logo=vite)

## ✨ Возможности

- 📷 **Захват с веб-камеры** — работает прямо в браузере
- 🧠 **ML распознавание** — определение 7 эмоций через face-api.js
- 🎨 **Динамический дизайн** — градиентный фон меняется в зависимости от эмоции
- 📱 **Адаптивность** — работает на десктопе и мобильных устройствах
- ⚡ **Без бэкенда** — все вычисления происходят локально

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/Emotes.git
cd Emotes

# Установить зависимости
npm install

# Запустить dev сервер
npm run dev
```

## 🛠 Технологии

- **React 19** — UI библиотека
- **TypeScript** — типизация
- **Vite** — сборщик
- **face-api.js** — распознавание лиц и эмоций (TensorFlow.js)

## 📁 Структура проекта

```
src/
├── components/
│   ├── EmotionDisplay.tsx   # Отображение эмоции
│   ├── LoadingScreen.tsx    # Экран загрузки
│   └── WebcamView.tsx       # Компонент камеры
├── hooks/
│   └── useFaceDetection.ts  # Хук распознавания
├── types/
│   └── emotions.ts          # Типы и конфигурация эмоций
└── App.tsx                  # Главный компонент
```
